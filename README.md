# Sweet Delights Bakery Website

## Overview
This is a multi-page website for **Sweet Delights Bakery** in Johannesburg.  
It showcases the bakery’s products, news, gallery, and contact options. The site includes interactive features, animations, and forms.

## Folder Structure
- **CSS/** – Contains all stylesheet files (e.g., `style.css`)  
- **JavaScript/** – Contains all JavaScript files (e.g., `script.js`) for interactive elements like modals, lightbox gallery, dynamic products, form validation, and the map.  
- **images/** – Contains all image assets used in the website.  
- **Documents/** – Any additional documents or resources.  
- **HTML Files** – Includes:
  - `Homepage.html` – Main page (can be renamed to `index.html` if you want it to be served by default)
  - `About us page.html`  
  - `Gallery.html`  
  - `News.html`  
  - `Contact us.html`  
  - `Enquiry.html`  

## Features
- Responsive navigation bar with hamburger menu for mobile devices.  
- Lightbox gallery for product images.  
- Dynamic homepage messages.  
- Searchable product list.  
- Interactive map using Leaflet.  
- Forms with client-side validation and mock AJAX submission.  
- Modal for news specials.  
- CSS animations and transitions for fade-ins and hover effects.

## Notes
- **Homepage:** Currently `Homepage.html`. If you want GitHub Pages to serve it automatically at the root URL, rename it to `index.html`.  
- All forms are validated using JavaScript before submission.  
- The website is fully static; email sending is simulated with JavaScript alerts.
